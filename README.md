Interactive Resume Builder
Overview
This is a static interactive resume built using HTML, CSS, and TypeScript as part of a hackathon project. The goal of the project is to showcase personal information, education, skills, and work experience in a clean and interactive format. The project is deployed using Vercel and can be accessed here.

Features
Personal Information: Name, email, and profile picture.
Education Section: Displaying educational background.
Skills Section: A list of skills with a show/hide toggle feature for interactivity.
Work Experience: Details of relevant work experience.
Static Deployment: Hosted live on Vercel for easy accessibility.
Technologies Used
HTML: Structure of the webpage.
CSS: Styling for a modern and clean layout.
TypeScript: Adds basic interactivity (like show/hide buttons).
Vercel: For deployment and hosting.
